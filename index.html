TUJUAN

Tujuan dari aplikasi Exit Survey Universitas Serambi Mekkah Banda Aceh ini adalah

Mendapatkan Data Lengkap Alumni

Aplikasi ini bertujuan untuk mendapatkan data lengkap para alumni Universitas Serambi Makkah.

Mendapatkan Umpan Balik

mendapatkan umpan balik mengenai proses pembelajaran pada Universitas Serambi Mekkah.

Peningkatan Mutu Universitas

meningkatkan mutu pendidikan di universitas kita yang tercinta ini. Data lulusan juga akan digunakan untuk melakukan Tracer Studi para alumni USM.

Prosedur Registrasi

Untuk menjamin bahwa data yang diisi oleh alumni adalah data yang sebenarnya, maka setiap calon alumni diharuskan untuk melakukan pendaftaran/registrasi terlebih dahulu.

1. Pencocokan Data

Pada proses pendaftaran akan dilakukan verifikasi data calon alumni dengan data yang tercatat pada database induk mahasiswa Universitas Serambi Mekkah

2. Registrasi

memilih menu "Registrasi" pada bagian menu yang berada di navbar bagian atas

3. Isi Biodata Nama

Selanjutnya calon alumni memasukkan data nama sesuai dengan yang tercatat pada akte kelahiran agar cocok dengan database induk

4. Isi Biodata NPM

Selanjutnya calon alumni akan diminta memasukkan NPM 13 digit. NPM ini akan digunakan untuk memverifikasi data dengan database induk agar tidak terjadi penyalahgunaan NPM.

5. Mengisi Email Aktif

selanjutnya wajib menggunakan email yang aktif karna akan digunakan untuk reset password. setelah mendaftar mahasiswa wajib menverifikasi email dengan cara klick verifikasi pada email masuk yang dikirim oleh aplikasi exit survey ini.

6. Membuat Password

Mahasiswa membuat password baru untuk login ke Aplikasi Exit Survey USM.

Cetak Bukti
Keikut Sertaan Survey

Setelah mengisi survey anda diharuskan melakukan pencetakan bukti dengan mengklick tombol "Cetak Bukti Exit Survey" pada bagian menu yang berada di bagian bawah setelah anda finish isi exit survey.

Hasil cetak menjadi bukti dan syarat administrasi yang Wajib dilampirkan dalam berkas pengusulan yudisium. Menu ini baru akan aktif jika Anda telah mengisi seluruh pertanyaan pada survey.
